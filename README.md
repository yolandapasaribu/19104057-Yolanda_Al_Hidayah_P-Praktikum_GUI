## Aplikasi GUI sederhana Data Mahasiswa 
Aplikasi GUI sederhana yang bertujuan untuk mendata mahasiswa terdiri dari fitur tambah, edit, tampilan data mahasiswa dan hapus data mahasiswa menggunakan PyQT5
